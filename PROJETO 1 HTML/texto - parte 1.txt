Curso Básico de Lógica de Programação com Python de Mulheres para Mulheres: Saindo do Zero
Atendendo a muitos pedidos da comunidade, Elas Programam e Quero Ser Programadora se uniram novamente para lançar mais uma turma do curso básico de lógica de programação com Python, exclusivo para mulheres.

Esse curso é para mulheres que desejam aprender a programar e precisam de ajuda nos primeiros passos. Se você já tentou aprender sozinha e sentiu muitas dificuldades, esse curso é para você.

Venha aprender lógica de programação em curso feito por mulheres para mulheres!
